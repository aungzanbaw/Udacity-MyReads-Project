# MyReads Udacity Project

MyReads is a simple project for React Fundamental session which is part of the React Nanodegree.

# Installation

Clone the repository or download the zip, to install the dependencies `npm install`
# Usage

Start the development server with
* Start the project `npm start`
* Build the project `npm build`
* Test the project `npm test --env=jsdom`

# Contributing

This repository is final assessment for Udacity students. Therefore, will not accept pull requests.
